### 心形粒子 (Heart Particle Animation)

一个基于手势控制的心形粒子动画系统。

**功能特性：**
- 🤚 手势识别：使用 MediaPipe 进行实时手势检测
- ❤️ 心形动画：握拳时粒子汇聚成均匀分布的立体心形
- 🎨 颜色过渡：心形时蓝色渐变为红色，散开时红色渐变为蓝色
- 🔄 动态效果：非心形状态下粒子会旋转移动
- 🎯 多种形状：支持球体、立方体、环形、DNA螺旋等多种粒子形状

**技术栈：**
- Three.js - 3D 图形渲染
- MediaPipe - 手势识别
- React - 用户界面
- WebGL - 硬件加速渲染

**使用方法：**
1. 打开 `心形粒子.html` 文件
2. 允许摄像头权限
3. 握拳看到心形，张开手看到粒子散开

**文件：** [心形粒子.html](./心形粒子.html)
